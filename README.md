LocalSearchEngine
=================

Perl implementation of local file-system searching engine

Prerequisite Perl Modules:
* DBM::Deep
* Tck::Tk
* Encode::Detect::Detector
* AudioFile::Info
* MP4::Info
* CAM::PDF
* Data::Dumper
* HTML::TreeBuilder
* Net::Ping::External
* Archive::Zip
* Digest::MD5
